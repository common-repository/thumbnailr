Recent images from your Thumbnailr site on your Wordpress blog.

== Description ==

This plugin shows recent images of your [Thumbnailr](http://www.thumbnailr.com/) site on your Wordpress blog. Nothing more and nother less.

Remember: suggestions are more than welcome!
Remember 2: **this plugin is still under heavy development**. Use it at your own risk.

== Installation ==

1. Go to **"Plugins"** > **"Add Plugin"**
2. Search for **"Thumbnailr"**
3. Click **"Install Now"**
4. Activate the plugin

Note to use this plugin you need to have Thumbnailr site, it's free and fun so no worries there.

A nice two step tutorial on how to install and configure is available at:
[Wordpress plugin installation manual](http://support.thumbnailr.com/pages/wordpress-plugin-installation-manual.html)
